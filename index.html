About the Project

⚠️ Status: This project is currently under active development. Features may change frequently.

Title : Neural Network for Object Detection in a self-driving car simulator.

Controls : Use Arrow Keys to Drive the Car Manually

Technologies Used : HTML, CSS, JavaScript

Description : This project simulates a self-driving car using a neural network to detect and avoid obstacles on the road. The car is trained using a genetic algorithm to improve its performance over time. The simulation includes a visual representation of the neural network and the car's path.
